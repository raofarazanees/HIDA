To set a token locally follow these steps:

  1. In a new tab, login with the Login MicroUI on snapshot
  2. Open up Developer Tools > Application > Storage > Local Storage
  3. Copy the entire value of ls.token
  4. Return to this tab with your app running locally and open up Developer Tools > Application > Storage > Local Storage
  5. Create a new "Key" for your token called ls.token in the "Key" field
  6. Paste the entire value of ls.token from snapshot in the "Value" field
  7. Refresh the page to see this take effect

If not local, the app should automatically redirect you to the Login MicroUI with the correct appId. (envrionment dependent)

For your application to correctly integrate with the Login Portal, please follow the steps for adding a new platform app.
